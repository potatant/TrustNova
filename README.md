# TrustNova
Overview
TrustNova is a machine learning-based solution designed to improve the loan approval process in the banking sector.
TrustNova addresses these challenges by implementing automated credit scoring models using classification algorithms like Logistic Regression, Decision Trees, and Random Forests. The system evaluates applicants based on both conventional features (credit score, income, loan amount) and non-traditional data (utility bills, mobile usage patterns), paving the way for fairer and more inclusive financial systems.

---Objectives
-Automate loan approval using ML classification models
-Enhance prediction accuracy for creditworthiness
-Reduce human bias and subjectivity
-Improve operational efficiency and decision-making speed
-Support financial inclusion for underbanked and low-income individuals

---Algorithms Used
-Logistic Regression
-Decision Tree Classifier
-Random Forest Classifier
Each model was trained on labeled loan application data and evaluated using metrics such as accuracy, precision, recall, and confusion matrices to compare their predictive power.

---Features include:
-Credit Score
-Applicant Income
-Loan Amount
-Employment Status

---Results & Impact
-Improved prediction accuracy over manual evaluation
-Reduced processing time and operational cost
-Increased loan approval rate for previously underserved applicants
-Transparent and explainable predictions using model interpretability tools
